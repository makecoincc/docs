### Solana Playground

[Solana Playground](https://beta.solpg.io/) is an online development environment provided by Solana. The advantage is that you don't have to install any programs to use solana cli, and you can also develop programs.

### Create Wallet

Click "Not connected" in the lower right corner of Solana Playground to create a wallet.

### Token Issuance via Command Line

To create tokens using Solana Playground's command line, please refer to [Deploy Tokens via Command Line](/en/docs/solana/create-token/create-token-cli)
