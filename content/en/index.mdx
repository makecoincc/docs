### This is a Token Issuance Platform

Digital assets are the core of blockchain, and everything in web3 revolves around them. This platform focuses on token issuance and currently only supports token issuance on Solana, with plans to support more chains in the future.

### No Fees

This platform is mainly for technical exchange and learning purposes and does not charge users.

### Open Source Code

This platform is mainly for technical exchange and learning purposes, and all code is open source. It focuses on web development technology, with TypeScript as the main language and Next.js as the main framework.

### Currently Only Supports Solana

Because token issuance on Solana is relatively simple, and the official website provides a large number of sample codes, making technical implementation convenient, we start from here.
