---
title: 设置权限
description: 更改或撤销 mint 和 token account 的权限
---

通过 SetAuthority 指令可以更改或撤销 mint 和 token account 的权限。只有当前权限持有者可以将其权限转移到新地址，或通过将权限设置为 null 来永久撤销。一旦撤销，权限将无法恢复。

### Mint Account 权限
- Mint Authority（铸造权限）：控制新代币的创建。可以将代币铸造到任何 token account。通常在初始供应创建后撤销，以创建固定供应的代币。
- Freeze Authority（冻结权限）：控制冻结和解冻 token account 的能力。可以阻止任何 token account 转移代币。通常会被撤销，以保证用户的代币不能被冻结。
### Token Account 权限
- Account Owner（账户所有者）：对 token account 拥有完全控制权。可以转移代币、销毁代币、批准代理人以及在余额为零时关闭账户。
- Close Authority（关闭权限）：可以在账户余额为零时关闭 token account。默认情况下，这是账户所有者，但可以委托给其他账户。
