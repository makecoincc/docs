### 批准委托
授予另一个账户（委托账户）从您的 token account 转移特定数量 token 的权限。委托账户可以转移的 token 数量不能超过批准的额度。
只有 token account 的所有者可以批准委托，并且每个 token account 同一时间只能有一个委托账户。任何新的批准都会覆盖之前的批准。

### 撤销委托
移除当前已批准委托的所有转账权限。撤销后，委托将无法再从您的账户中转移任何代币。只有 token account 的所有者可以撤销委托。
